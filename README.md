# dprint

Print specified values from desktop files to stdout.

## Usage

`dprint [-v] [-d path] [-i key:val] [-o key]`

Any input value can be replaced with `-` to use the value from stdin instead.

## Building

Install the dependencies:

- go (>=1.12)
- scdoc

Then compile dprint:

    $ make

## Installation

    # make install
